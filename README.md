# emberjs-rails

Ember.js for Rails 3.1+

This gem provides Ember.js 0.9.3

## Usage

The ususal asset pipeline biz-nass:

    //= require emberjs

Or, an uncompressed version:

    //= require emberjs-dev

### Installation

Add this to your Gemfile:

    gem "emberjs-rails"

Done. Go forth, grasshopper.
